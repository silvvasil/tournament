# tournament
This was a strategy's tournament in Vzlet school. 17-23 February 2024

# Try it yourself

```
python3 manage.py runserver 127.0.0.1:8000
```
and then check `127.0.0.1:8000/results`

you can also upload some strategies from `strategies.zip` to `127.0.0.1:8000/upload`

cells of results table are clickable - you will be redirected `127.0.0.1:8000/battle/id_1/id_2` to view the battle between users
